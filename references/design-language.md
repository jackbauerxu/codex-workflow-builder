# Design Language and Style Lock

## 不可漂移的基础规则

- 默认交付为 1080×1920、9:16、30fps；除非平台或用户明确指定。
- 每屏只保留一个阅读目标：标题、主体、数据或动作。
- 标题安全区在上 16%，字幕安全区在下 18%；主体不得被字幕覆盖。
- 一个场景最多两级文字、一个强调色、一个转场动机。
- 所有样式来自 `styles/` 的一个预设；预设改变时必须在 Production Configuration 里记录。

## Style Lock

选择预设后，字体、配色、网格、字幕、镜头速度和转场只能由该预设定义。**No generic-style override**：其它设计 Skill、临时灵感或无来源参考不能静默混入第二套设计语言。只有用户明确请求“将预设从 X 改为 Y”时才允许更换，并重新审查所有场景。

## 质量底线

精致不等于堆纹理、玻璃态、粒子、镜头光晕或动效。若一个视觉元素不能解释内容、建立层级或引导下一步，就删除它。
