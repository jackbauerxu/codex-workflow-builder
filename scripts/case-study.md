# 案例复盘项目脚本

适用于已有授权、可核实过程资料的项目复盘或客户案例。重点是可迁移的决策，不是把个案包装成普遍结果。

输入：`context`、`constraint`、`intervention`、`evidence[]`、`outcome`、`limitations`。

| 节拍 | 内容 |
| --- | --- |
| 起点 | 谁在什么条件下遇到什么问题。 |
| 限制 | 时间、资源、权限或素材的真实约束。 |
| 关键决策 | 只选一个改变结果的选择，展示理由。 |
| 证据 | 前后状态、过程记录或已授权数据。 |
| 可迁移部分 | 哪个方法可借鉴，哪些条件不可照搬。 |

输出字段：`case_context`、`constraint`、`decision`、`evidence_refs[]`、`outcome`、`non_generalizable_limit`。未获授权的客户名、截图或结果必须匿名或移除。
