# 方法与教程项目脚本

适用于观众需要跟着完成一个具体结果的教程。先给终点，再给最少步骤，并让失败条件可见。

输入：`starting_state`、`end_state`、`steps[]`、`prerequisites`、`failure_mode`。

1. **可见结果**：开场同时出现起点与终点，明确适用前提。
2. **最小路径**：仅保留 2-4 步；每步包含动作、界面/物件证据和完成信号。
3. **常见卡点**：展示一个容易失败的分支与纠正方式。
4. **自检**：告诉观众怎样验证自己是否完成。
5. **保存 CTA**：提示需要时回看具体步骤，不添加第二个诉求。

输出字段：`starting_state`、`end_state`、`steps[]`、`success_signal`、`failure_mode`、`save_cta`。不把前置条件藏在字幕小字里。
