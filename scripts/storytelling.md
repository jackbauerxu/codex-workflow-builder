# 叙事脚本模板

适用于 15-60 秒的解释型视频。采用四拍结构，而不是把文章逐句念出来。

1. **问题**（0-20%）：观众正在面对的具体摩擦或误解。
2. **转折**（20-40%）：指出旧方式失效的原因，用一个证据、界面或对照承接。
3. **方法**（40-80%）：只讲 2-3 个可执行动作；每个动作配一个镜头或 B-roll。
4. **落点**（80-100%）：给判断、下一步或收藏理由，不重复开场。

每一拍写成：`narration`、`on_screen_text`、`shot_id`、`duration_seconds`、`fact_source`。没有事实来源的数字、客户结果和新闻结论不能进入旁白或字幕。
