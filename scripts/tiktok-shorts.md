# TikTok / Shorts 爆点项目脚本

适用于技巧、复盘、挑战、鲜明观点或轻量教程。用一个冲突带动完整观点，不把快节奏误解为高密度噪音。

输入：`audience_tension`、`verified_takeaway`、`proof_or_demo`、`one_action`。

| 节拍 | 时长 | 写法 |
| --- | --- | --- |
| 停留理由 | 0-2 秒 | 反常识、现场问题或结果先行，三选一。 |
| 立场 | 2-5 秒 | 只给一个判断，不预告十个观点。 |
| 证明 | 5-18 秒 | 录屏、对照、过程或来源；每 1-2.5 秒切一次有意义的镜头。 |
| 动作 | 18-25 秒 | 给一个当下可做的步骤。 |
| 回收 | 25-30 秒 | 呼应开场，用一个 CTA 收尾。 |

输出字段：`scroll_stop_line`、`claim`、`proof_beat`、`action_step`、`callback`、`cta`。禁止虚构“人人都在用”“立刻翻倍”等不可证实的社会证明或效果。
