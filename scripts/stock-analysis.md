# 股票分析脚本模板

适用于基于用户提供财报、公告、公开披露或明确日期数据的 30-60 秒解释视频；不是荐股、价格预测或交易指令。

## 五拍结构

1. **事实钩子**：`截至 {as_of_date}，{company} 刚披露了 {verified_event}。` 只放一个可核验数字或事件。
2. **业务驱动**：说明 1-2 个已披露的收入、成本、产品、用户或行业变量；每项绑定来源和口径。
3. **关键数据**：一张数据卡或图表。写清单位、期间、同比/环比口径和来源；不能用动画改变数值含义。
4. **风险与未知项**：列出一个已披露风险、估值不确定性或尚未确认的问题；不把观点说成事实。
5. **收尾**：`这不是投资建议；下一步应核对 {source_or_metric}。` 不给买卖指令、目标价或保证收益。

每一拍返回：`narration`、`on_screen_text`、`shot_id`、`duration_seconds`、`as_of_date`、`fact_source`、`data_unit`、`qualification`。任何字段缺失时标为 `insufficient_data`，而不是以预测或虚构图表补全。
