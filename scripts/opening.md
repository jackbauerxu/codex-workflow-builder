# 开场脚本模板

适用于 0-3 秒。每条只选一种开场，不叠加提问、数字和结论。

| 类型 | 公式 | 画面 |
| --- | --- | --- |
| 反常识 | `你以为 ___，真正卡住的是 ___。` | 先给旧做法，再切到关键变量。 |
| 结果先行 | `我用 ___，把 ___ 从 ___ 变成 ___。` | 只使用已证实的结果或定性描述。 |
| 现场问题 | `这个画面里，___ 正在浪费你的时间。` | 录屏、物件或真实场景特写。 |
| 新闻钩子 | `今天这件事，影响的不是 ___，而是 ___。` | 日期/来源卡 + 一个关键实体。 |

输出字段：`hook_line`、`proof_or_qualification`、`opening_shot_id`、`on_screen_text`。字幕不超过两行，每行不超过 13 个汉字。
