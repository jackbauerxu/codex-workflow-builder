# AI 产品演示提示词

```text
为 {product_name} 的 {feature} 做 {duration_seconds} 秒 9:16 演示。
用户起点：{before_state}；完成后可验证状态：{after_state}。
用 {style_preset}，脚本采用问题 → 录屏动作 → 可观察结果 → 单一 CTA。
镜头至少包含 hook-problem、screen-proof、feature-detail、result-state；不要虚构速度、准确率、用户数或客户案例。
输出每镜的录屏区域、光标动作、字幕、时长和需要的真实素材路径。
```

适合工具、SaaS、自动化流程和界面更新；优先真实录屏，不把虚构 UI 当作产品证明。
