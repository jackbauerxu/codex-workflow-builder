# 开箱视频提示词

```text
为 {product_name} 设计 {duration_seconds} 秒 9:16 开箱视频。
观众：{audience}；可展示的真实包装、物件、配件与使用场景：{available_assets}。
选用 {style_preset}，按 包装承诺 → 拆封动作 → 一个关键细节 → 首次使用 → 单一 CTA 排镜。
每镜写 shot_id、手部动作、景别、真实素材路径、字幕、时长和不可声称事项。
不展示不存在的配件、价格、抽奖、联名或夸张结果；素材缺失时标记 missing_asset。
```

开箱的价值在真实物件与使用细节，不以撕包装、彩纸或快速蒙太奇替代产品信息。
